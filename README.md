# СоцОпрос

Анонимный социальный опросник для пенсионеров. Квиз → рекомендации → реф-ссылки на Т-Банк и Альфа-Банк.

## Локально

```bash
cd soc-opros
npm run dev
```

Откроется на http://localhost:3456

## Деплой на Vercel

1. `vercel` из папки `soc-opros`
2. Привязать домен `soc-opros.vercel.app`
3. В `app.js` заменить `METRIKA_ID` на ID счётчика Яндекс.Метрики
